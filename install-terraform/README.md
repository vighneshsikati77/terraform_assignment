# Install Terraform

## Objective
Install Terraform on the local machine and verify that it is working correctly.

## Steps Performed
1. Downloaded Terraform from the official HashiCorp website.
2. Extracted the Terraform executable.
3. Added Terraform to the system PATH.
4. Verified the installation using the command:
   ```bash
   terraform version
   ```
5. Installed AWS CLI.
6. Configured AWS CLI using:
   ```bash
   aws configure
   ```

## Tools Used
- Terraform
- AWS CLI
- Windows/Linux Terminal

## Key Learning
- Learned how to install Terraform.
- Configured AWS credentials.
- Verified Terraform installation successfully.

## Outcome
Terraform and AWS CLI were successfully installed and configured.
