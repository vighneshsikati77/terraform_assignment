# What is Infrastructure as Code (IaC)

## Objective
Understand the concept of Infrastructure as Code (IaC) and how Terraform automates infrastructure provisioning.

## Topics Covered
- Introduction to Infrastructure as Code
- Benefits of IaC
- Terraform Overview
- Declarative vs Imperative Infrastructure
- Terraform Workflow

## Key Learning
- Infrastructure can be managed using code instead of manual configuration.
- Terraform uses configuration files to provision and manage cloud resources.
- IaC improves consistency, automation, scalability, and version control.

## Tools Used
- Terraform
- AWS (Conceptual)

## Outcome
Successfully understood the fundamentals of Infrastructure as Code and the Terraform workflow.
